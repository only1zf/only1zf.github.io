4.6 第四道分叉：强重叠课程（切换成本）​

  • 跨全部 12 专业必修：概率论与数理统计、数理方程（A 或 B）、大学物理 - 综合实验（A 或 B）
  • 物院 ↔ 核院切换最低成本：A 套基础一致；核 - 应物 ≈ 物 - 应物 + 几门工程附加
  • 工院内部 5 专业切换最低成本：B 套 + 理力 B + 材力 A + 工程计算方法 共通
  • 物院 ↔ 工院切换最高成本：从 A 套换 B 套需"回炉"；学院通修阶段的力学 A、电磁学 A 缓冲了部分

4.7 跨专业可选（少院特权 16 学分）​

学科交叉选修 16 学分可跨学院选——少院相对其他学院的最大灵活性来源：

  • 选物 - 应物 → 仍可选 机器人学导论 + 机器视觉与图像处理 + 信号与系统 B + 机器人设计与制作（补"动手"）
  • 选工 - 机器人 → 仍可选 光学 A + 原子物理 A + 量子力学 A（补"物理深度"）
  • 选核 - 工物 → 已含等离子体物理导论/受控热核聚变导论；想保留 CS 接口可选 数据结构 B（自由选修）

4.8 命名碰撞警示​

"应用物理学" 物院 21102 与核院 21403 同名但课表/方向均不同：

  • 物院 21102 = 凝聚态/物理电子/微电固体/生物物理/等离子体物理/医学物理（方向群在物院 4 个系内）
  • 核院 21403 = 大装置光学科学/等离子体物理（依托大科学装置）
  • 两者共有"等离子体物理"方向——不仅名字相同，方向上也有重叠；选 21102 也可走等离子体路线
  • 报志愿时务必看专业代码而非专业名

  • 物 - 应物（40，🥇）：热情 4（应用味、与"中量动手"匹配、不绑死前沿理论）；灵活 4（数理深 + 偏应用，转半导体/光电/CS 都容易）；就业 4（产业出口多）；学术 4（USTC 物院传统强项）

  • 工 - 力学（37）：3 院士（伍小平/杜善义/刘果）+ 陆夕云杰青；与中量动手匹配；物院 A 套最近的工院专业，回切物理成本最低

  • 核 - 工物（35）：双万一流、大装置入门券；学术好，但热情未必比物 - 应物高
  • 工 - 机器人（30）：就业风口好但重度动手（车间/烙铁/PCB/机械装调）与"中量动手"偏好错位
  • 核 - 核工（20）：灵活性最低（行业垂直、体制内），与"灵活性权重 3"严重失配；仍保留为候选

八、5 个候选专业的入选理由​

🥇 物 - 应用物理学（40，默认主修）​

加权第一。可逆性最大：与物 - 物理学共表 → 大三可走理论；含光电方向 → 大三可专攻光电；与核 - 应物相近 → 大三申请跨院难度可控。灵活性 4：数理深 + 偏应用，转半导体/光电/CS/量化金融都有路径。与画像 100% 匹配：中量动手（物理实验 + 仿真）+ 不锁前沿（应物方向群中可选不前沿的）+ 不出国。

🥈 核 - 应用物理学（核）（38）​

与物 - 应物极相近（课表 80% 重叠）+ 大装置入门券（合肥光源/上海光源/等离子）。减分点：灵活性略低（行业窄一点，但仍可转半导体）。建议交叉选修：光学 A + 原子物理 A，可弥补物理深度。

🥉 物 - 光电信息科学与工程（37）​

双万计划国家级一流本科专业。方向具体：光刻机/光通信/激光/光子医学。减分点：锁定光电方向（不像物 - 应物保留多方向可能）。建议交叉选修：近代光学基础 + 激光原理与技术。

工 - 理论与应用力学（37，重评后上调）​

工院师资能见度最高：3 院士（伍小平/杜善义/刘果）+ 杰青陆夕云。物院 A 套装最近的工院专业：理论力学 B + 数理方程 A + 大学物理 - 综合实验 B → 回切物理路径成本最低。与中量动手完全匹配（湍流/CFD 仿真 + 实验力学测量）。出口明确：航天/航空/车企/中科院力学所。

物 - 物理学（36）​

灵活性最强 5（数理深、转码/转量化/读 PhD 通吃）。学术 5（USTC 王牌）。减分关键：热情 2 分——其卖点（量子信息/超导/低维拓扑/超冷原子）正是用户"不主动追求"的方向。注：与物 - 应物共表，大三再决定理论 vs 应用即可。

工 - 能源与动力工程（36，重评后上调）​

师资豪华：1 本院士（舒歌群） + 吴宜灿核能聚变院士 + 3 兼职院士。核能/聚变赛道第二入口（不必走核学院）。三条出口：航空发动机（中国商发）/ 新能源车（动力电池/电控）/ 聚变能（ITER 路径）。与中量动手匹配（实验台架 + 仿真）。

6.5 重要发现：核能/聚变方向不止在核学院​

吴宜灿院士（中科院核能安全技术研究所所长，ITER 核安全专家组委员）挂在工院热能系——这意味着：

  • 选工 - 能源动力也可走核能/聚变赛道（铅基堆方向、ITER）
  • 选核 - 工程物理走聚变科学与技术方向（受控热核聚变导论必修）
  • 这两条路径都通向 CFETR / EAST / BEST 等大装置

→ 对工 - 能源的"个人热情"打分上调 1 分（如果用户对核能/聚变感兴趣）。

关系说明
物 - 应物 ≡ 物 - 物理学（课表层面）专业基础课完全相同（共用一张课表），差别仅在大三的交叉选修建议——这两者的决策可延迟到大三（少院规则允许"大三在修读院系内自主调整专业"）
物 - 应物 ⊇ 物 - 光电（方向层面）光电是应物的子集方向。选物 - 应物保留转光电的全部能力；反之锁定
物 - 应物 ↔ 核 - 应物唯一不可逆决策——分属不同学院，需现在决定

→ 真正的二选一只是：入物院 还是 入核院 还是 入工院。

十一、默认主修建议与对比理由​

11.1 默认主修：物 - 应用物理学（21102）​

  1. 加权第一（40 分）；与"中量动手 + 不主动追前沿 + 高灵活性"画像最契合
  2. 可逆性最大：大三可在物院内调整到物理学/光电；大四可全校选修跨院课
  3. 与 top 6 其他 5 个的"延迟决策"成本最低
  4. 与"少院走一步看一步"偏好契合

11.2 为什么不是物 - 物理学​

  • 课表层面完全相同（专业基础课共用）→ 大三再决定理论 vs 应用即可
  • 物 - 物理学的热情扣分（前沿物理）拖低总分；物 - 应物没有这个绑定
  • 物 - 应物的"方向群"更宽（凝聚态/物理电子/微电固体/生物物理/等离子/医学物理）→ 给大二大三留更多探索空间

11.3 为什么不是核 - 应物​

  • 灵活性高 1 分（行业出口比核学院宽）
  • 不出国前提下，物 - 应物对接半导体/光电/CS/金融都比核 - 应物路径多
  • 核 - 应物的"大装置入门券"对用户没有专属吸引力（用户对前沿不主动追）

11.4 为什么不是工 - 力学 / 工 - 能源​

  • 加权差 3-4 分
  • 工院两个备胎在"师资可见度 + 就业出口明确"上有优势，但灵活性 + 物理深度输物 - 应物
  • 工院作备胎而非主推；工院二选一留到分流前再决定
